# NucbaApi

## Instalaciones 
- `npm init -y`
- `npm i express`
- `npm i typescript`
- `tsc --init` → actualizar el tscongif (outDir)
- `npm i nodemon` 
- `tsc --watch`
- `nodemon ./dist/app.js` → Si no funciona probar con `npm install -g nodemon`
- `npm i --save-dev @types/express`
- Agregar: `"type": "module",` en el package.json
- `npm i dotenv` → Variables de entorno
- `npm i mongoose`
- `npm i cors`  → asegura que nadie que yo no quiera se conecte a mi api
- `npm i --save-dev @types/cors`
- `npm i jsonwebtoken`
- `npm i --save-dev @types/jsonwebtoken`
- `npm i bcryptjs` → Encriptación
- `npm i randomstring` →  genera codigo aleatorio
- `npm i --save-dev @types/randomstring`
- `npm i express-validator`
